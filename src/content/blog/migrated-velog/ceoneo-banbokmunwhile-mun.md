여러분, 이제 프로그램이 **“조건에 따라서 여러 번 같은 동작”**을 하게 만들 차례입니다.
예를 들어 “치킨을 배달 올 때까지 계속 기다린다” 같은 상황이 있겠죠.
한 번만 기다리는 게 아니라, 조건이 만족될 때까지 계속 반복하는 거예요.
이럴 때 쓰는 게 바로 `while`문입니다.

## while문 기본 구조

```c
while (조건식) {
    // 조건식이 참(1)인 동안 반복 실행할 코드
}
```

조건식이 **참**일 때는 안에 있는 코드를 계속 반복합니다.
조건식이 **거짓(0)**이 되면 반복을 멈추고 빠져나옵니다.

## 예시: 1부터 5까지 출력하기

```c
#include <stdio.h>

int main() {
    int i = 1;

    while (i <= 5) {
        printf("%d\n", i);
        i++;  // i를 1씩 증가시킴
    }

    return 0;
}
```

실행 결과:

```
1
2
3
4
5
```

여기서 핵심은 `i++` 부분이에요.
이걸 안 해주면 `i`가 계속 `1`이라서 조건식이 영원히 참 → **무한 루프**가 됩니다.
> 혹시나 잊었을까봐 한번 더 설명하자면 `i++`는 `1`씩 증가 시키는 코드 입니다.
계속 증가하다가 `while (i <= 5)` 조건이 참이 되면(즉, `i`가 `5`가 되면) `while`문을 빠져나와서 반복이 멈춥니다. 


### 무한 루프가 뭐임?

조건이 항상 참이라면 `while`문은 멈추지 않고 계속 실행됩니다.
여러분이 치킨을 평생 기다리는 것과 같아요.
웹으로 따지면 무한 로딩이라고 생각하면 됨.

예를 들어 이런 코드

```c
while (1) {
    printf("멈추지 않고 계속 출력해서 여러분 컴퓨터 고장낼거임\n");
}
```

이건 영원히 멈추지 않습니다.
이거 코드 안에 작성하고 실행시키지마세요. 여러분 컴퓨터 사양 안좋으면 터질수도 있음 ㄹㅇ임
궁금하면 해보세요 ***책임안짐**

위와 같은 이유 때문에 실제로는 `break;` 나 조건문을 넣어서 중간에 빠져나가게 만들어야 합니다.

## break와 continue
- `break;` : 반복문을 강제로 종료
- `continue;` : 이번 반복만 건너뛰고 다음 반복으로 넘어감

```c
#include <stdio.h>

int main() {
    int i = 0;

    while (i < 10) {
        i++;

        if (i == 3) continue;  // 3은 건너뜀
        if (i == 8) break;     // 8에서 멈춤

        printf("%d\n", i);
    }

    return 0;
}
```

실행 결과:

```
1
2
4
5
6
7
```

- 3은 `continue` 때문에 출력 안 됨
- 8에서 `break` 만나고 반복 종료

---

## while문 정리
- `while (조건)` → 조건이 참인 동안 계속 반복
- 변수 증감이 없으면 무한 루프에 빠질 수 있음
- `break`로 반복 중간에 끊을 수 있고, `continue`로 특정 조건만 건너뛸 수 있음

---

다음은 `for`문을 배워보겠습니다.
`while`문이 “**조건** 기반 반복”이라면, `for`문은 “**횟수** 기반 반복”에 훨씬 강력합니다.
