여러분, 이제 단순히 계산만 하는 건 재미없잖아요.
프로그래밍이 진짜 재미있어지는 지점은, 상황에 따라 다른 행동을 할 때입니다.
바로 조건문이 그걸 해줍니다.

**“돈이 0보다 크면 치킨 시키고, 아니면 라면 먹는다.”**
이런 상황을 코드로 표현하는 게 조건문이에요.

## if문 기본 구조
```c
if (조건식) {
    // 조건식이 참일 때 실행되는 코드
}
```

조건식이 **참(1)**이면 중괄호 { } 안의 코드가 실행되고,
조건식이 **거짓(0)**이면 그냥 넘어갑니다.

나는 돈이 **10,000원** 있는데 만약 돈이 **0원** 보다 많으면 치킨을 시킬거에요.
그 조건을 코드로 작성한겁니다.

```c
#include <stdio.h>

int main() {
    int 돈 = 10000;

    if (돈 > 0) {
        printf("치킨을 시킵니다.\n");
    }

    return 0;
}
```

실행 결과:

```
치킨을 시킵니다.
```

## if-else문

조건이 참일 때와 거짓일 때를 나눠서 각각 다른 행동을 할 수 있습니다.

```c
if (조건식) {
    // 참일 때 실행
} else {
    // 거짓일 때 실행
}
```

위에서는 우리가 **돈이 0보다 많으면** 치킨을 시키는거 밖에 없었잖아요.
이번에는 **돈이 0보다 많지 않다면** 이라는 조건을 추가한거에요.
모 아니면 도 인거죠.

예시:

```c
#include <stdio.h>

int main() {
    int 돈 = 0;

    if (돈 > 0) {
        printf("치킨을 시킵니다.\n");
    } else {
        printf("라면을 끓입니다.\n");
    }

    return 0;
}
```

실행 결과:

```
라면을 끓입니다.
```

## else if문

조건이 여러 개일 때는 else if를 이어서 씁니다.
예를 들어 돈의 액수에 따라 메뉴가 달라지게 해봅시다.

우리가 **5,000원**이 있고 돈이 **딱 10,000원이 있거나 그보다 더 많으면** 치킨을 시킬거고
돈이 **딱 5,000원 있거나 그보다 더 많으면** 피자를 시킬껍니다.
그것 조차도 없으면 라면이나 끓여야죠 뭐.

```c
#include <stdio.h>

int main() {
    int 돈 = 5000;

    if (돈 >= 10000) {
        printf("치킨을 시킵니다.\n");
    } else if (돈 >= 5000) {
        printf("피자를 시킵니다.\n");
    } else {
        printf("라면을 끓입니다.\n");
    }

    return 0;
}
```

실행 결과:

```
피자를 시킵니다.
```

## 비교 연산자와 조건문

아까 위에서 본 수학적 기호 있죠? 이건 **비교 연산자**에요. 수포자라도 이정도는 알지않을까요?
**조건문은 비교 연산자와 거의 세트로 사용됩니다.**
- `==` : 같다
- `!=` : 다르다
- `>` : 크다
- `<` : 작다
- `>=` : 크거나 같다
- `<=` : 작거나 같다

위에서 이미 봤으니 간단하게 나이에 따라 성인인지 미성년자인지 알아보는 코드를 작성해봅시다.
결과를 예상해보세요.

```c
int 나이 = 20;

if (나이 >= 18) {
    printf("성인입니다.\n");
} else {
    printf("미성년자입니다.\n");
}
```


## 논리 연산자와 조건문

논리 연산자는 비교 연산자를 여러개써서 비교하고싶을 때 사용합니다.
- `&&` : AND 연산 (둘 다 참) *직역: 그리고
- `||` : OR 연산 (둘 중 하나라도 참) *직역: 또는
- `!` : NOT 연산 (참 → 거짓, 거짓 → 참)  *반대로 바꿔줌

우리가 위에서 내가 가지고 있는 돈과 나이를 가지고 조건을 써봤잖아요.
이거를 이용해서 술집에 들어갈 자격이 있는지 확인해보는 코드를 짤거에요.

나이가 `20살`이고 돈이 `10,000`원이 있습니다.
근데 술집 입장료가 `5,000원`이에요.
그리고 술집은 `성인`만 갈 수 있겠죠?


```c
int 나이 = 20;
int 돈 = 10000;

if (나이 >= 19 && 돈 >= 5000) {
    printf("너 성인인거 ㅇㅋ 입장료도 ㅇㅋ 술집 입장하셈\n");
}

if (나이 < 19 || 돈 >= 20000) {
    printf("술집에 들어갈 돈은 있지만 너 미성년자라서 안됨\n");
}

if (!(나이 >= 19)) {
    printf("그냥 미성년자라서 안됨\n");
}
```
여기서 마지막 조건에 보면 `!(나이 >= 19)` 이 조건 이해했나요?
앞에 `!` 만 빼고보면 사실상 `20 >= 19` 라서 술집 입장 가능한 나이잖아요
근데 `!`가 있음으로써 그 **반대**가 됩니다. 즉, 술집 입장 가능한 나이가 아니라는 거에요 
그냥 `if (나이 < 19)` 이렇게 써도 틀린 조건은 아닙니다.
근데 배웠잖아요. `NOT 연산` 한번 써보는거죠


---

## 오늘 정리
- `if (조건)` → 조건이 참일 때만 코드 실행
- `if-else` → 조건이 참/거짓일 때 각각 다른 코드 실행
- `else if` → 조건 여러 개 처리 가능
- **비교 연산자**, **논리 연산자**와 함께 자주 사용
