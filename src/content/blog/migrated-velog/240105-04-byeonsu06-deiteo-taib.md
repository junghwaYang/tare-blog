# 1. 변수란?
어떠한 값을 담기위한 데이터 공간 그 자체
> ### notice
- 값을 저장하기 위해 확보한 메모리 공간 자체
- 메모리 공간을 식별하기 위해 붙인 이름

# 2. 식별자란?
어떤 값을 식별하기위한 이름

# 3. 변수 선언 키워드 3가지
var, let, const

# 4. 변수 선언의 단계
1. 선언
~~2. 할당~~
***
> ### notice
- 변수 선언
- 변수 초기화


# 5. 아래 console.log의 결과값이 undefined가 나오는 이유는?
var의 경우는 할당을 별도로 하지않아도 undefined이란 값이 담기며,
변수 호이스팅에 의해 순차적으로 실행되는 런타임 이전에 실행되기 때문이다.

```jsx
console.log(score); // undefined

var score;
```

# 6. 호이스팅이란?
런타임 이전에 실행되어, 선언 전 최상단으로 이동하여 실행되는 것

# 7. 변수의 재할당이 불가능한 키워드는? 재할당이 불가능한 이유도 함께 작성
const
const는 상수 값을 담는 변수이며, 상수는 변하지않는 값이기에 재할당이 불가능하다.
***
> ### notice
- const는 단 한번만 할당 할 수 있는 변수이다.

# 13. 동적 타입 언어와 정적 타입 언어의 차이를 작성하고, 동적 타입 언어의 장,단점을 서술하시오
정적 타입은 변수 선언 시 데이터 타입을 지정해야만 선언이 가능하며, 재할당을 통해 데이터 타입 변경이 불가능하다.
동적 타입은 변수 선언 시 데이터 타입을 지정하지않아도 되는 유연성이 있지만 재할당을 통해 데이터 타입 변경이 가능하기때문에 신뢰성이 떨어진다.
