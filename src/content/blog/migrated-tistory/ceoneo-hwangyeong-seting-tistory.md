### 개발 시작 전에, 일단 준비물부터 챙기시죠

자, 여러분. 개발을 시작하려면 일단 기본 세팅부터 해둬야죠? 밥 먹기 전에 숟가락, 젓가락 챙기는 것처럼 말이에요. 개발할 때도 필요한 도구들이 있습니다.

뭘 세팅할 거냐면... 일단 컴퓨터는 0과 1밖에 모르는 바보라서 C언어 코드를 바로 이해하지 못해요. 그래서 사람이 쓴 코드를 컴퓨터가 알아들을 수 있도록 **번역기** 역할을 하는 게 **컴파일러**라고 합니다.

쉽게 말해, 우리가 한국말을 하는데 외국인이 와서 영어로 말하면 못 알아듣잖아요? 그럼 뭐 써요? 번역기 쓰죠? 그거랑 똑같습니다. 컴퓨터는 오직 기계어만 할 줄 알아요.

우리는 컴퓨터보다 하등생물이기 때문에 '을'의 위치에서 컴퓨터가 알아들을 수 있도록 기계어로 변환해서 말을 해야 하는데, 언제 0101101101111... 이렇게 치고 있을 거예요? 그걸 그냥 **컴파일러**가 알아서 해주는 겁니다.

근데 번역기만 있으면 뭐 해요? 번역기를 돌려주는 도구가 있어야 하잖아요. 그게 바로 **IDE**라고 하는데, 스타벅스 같은 데 가면 맥북에 막 까만 화면에 알록달록 코드 적혀있는 화면들 보셨죠? 그거 아마 **VS Code**나 **비주얼 스튜디오** 같은 거 썼을 겁니다.

이 강의에서는 번지르르하게 대단한 걸 만들지 않을 거라, 웹 개발자들이 많이 쓰는 VS Code를 사용할 거예요. 다른 IDE로는 비주얼 스튜디오가 있는데, 사실 C언어 개발할 때는 비주얼 스튜디오가 더 좋긴 하지만 딱히 상관은 없습니다. 개발 경험 좀 있으시면 아무거나 쓰세요.

### 1단계: 번역기(컴파일러) 설치

자, 그럼 우리 컴퓨터에 C언어 번역기부터 깔아봅시다.

윈도우 쓰는 분들은 [MinGW](https://sourceforge.net/projects/mingw/)라는 걸 설치하면 돼요. 구글에 **MinGW** 검색해서 들어가세요. 다운로드 버튼 누르고 설치 파일 받아서 다음, 다음, 다음 누르면 끝납니다. **설치할 때 gcc랑 g++ 옵션 체크하는 거 잊지 마시고요.**

### 2단계: 개발 도구(VS Code) 설치

다음으로 번역기를 돌릴 도구를 설치해야죠.

구글에 [Visual Studio Code](https://code.visualstudio.com/) 검색해서 다운로드하세요. 자기 운영체제에 맞는 걸로 다운받고, 이것도 그냥 다음, 다음, 다음 눌러서 설치하면 됩니다.

### 3단계: VS Code에서 번역기 쓰기

이제 도구랑 번역기를 연결해 줄 차례예요.

VS Code를 열고, 왼쪽 확장 탭으로 가서 [C/C++ 확장팩](https://marketplace.visualstudio.com/items?itemName=ms-vscode.cpptools)을 설치하세요. 이거 깔아야 VS Code에서 C언어 코드를 실행시킬 수 있습니다.

그리고 새로운 파일을 만들고 파일명 뒤에 .c를 붙여서 저장하세요. 예를 들어 first.c처럼요.

### 4단계: 시작하기

자, 이제 준비 끝났으니 첫 번째 코드 한 번 짜 봅시다.

`printf`가 뭔지, `int`가 뭔지 이런 건 일단 신경 끄고, 그냥 아래 코드를 그대로 따라 쳐 보세요.

```
#include 

int main() {
    printf("Hello, world!");
    return 0;
}
```

이제 VS Code에서 터미널을 열고 (Ctrl + `) 아래 명령어를 입력하고 엔터를 치세요. (맥os도 동일)

```
gcc first.c -o first
```

그럼 first.exe 파일이 생길 겁니다. 이제 이 파일을 실행시키면 돼요.

```
./first
```

엔터를 누르면 터미널에 Hello, world!가 찍힐 겁니다. 이 순간 여러분은 개발자가 된 거예요. 축하드립니다.

여기서 팁 하나 드리면 확장 탭에 [C/C++ Compile Run](https://marketplace.visualstudio.com/items?itemName=danielpinto8zz6.c-cpp-compile-run) 도 설치하세요.

이게 뭐냐구요? 위에 gcc first 어쩌고저쩌고 저거 말고 **F6 단축키** 딸깍으로 컴파일해주는 아주 아름다운 플러그인이에요.

개발할 때 이런 유용한 플러그인 잘 활용하면 엄청 편해집니다.

이제 기본적인 세팅은 끝났습니다. 다음 강의부터는 C언어 문법이 뭔지, 변수가 뭔지 하나씩 알아봅시다.
