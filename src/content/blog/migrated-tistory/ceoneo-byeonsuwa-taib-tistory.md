자, 여러분. 드디어 C언어 문법의 첫걸음을 떼는 순간이 왔습니다.

프로그래밍에서 제일 많이 하는 게 뭐냐? 바로 **저장**이에요.

생각해보세요. 우리도 평소에 뭔가 담을 때 상자 쓰잖아요.

반지는 반지 케이스에, 신발은 신발장에...

그래서 프로그래밍에서도 똑같습니다.

**데이터 = 반지**

**변수 = 반지 케이스**

그냥 간단하게 말하면, **변수는 데이터를 담는 상자**예요.

#### 변수, 어떻게 만드나요?

상자 하나 사려면 종류도 고르고, 이름표도 붙여야죠.

C언어에서 변수도 똑같습니다.

- **종류(타입)** &rarr; 뭘 담을 수 있는 상자인지

- **이름** &rarr; 나중에 꺼낼 때 부를 이름

```
종류 이름;        // 상자만 준비
종류 이름 = 값;   // 준비하면서 값까지 넣기

int age;          // age라는 상자만 만들었음
int age = 20;     // 만들면서 20이라는 값도 같이 담았음
```

이렇게 두 가지 방식으로 변수 선언이 가능합니다.

## 정수형 변수 (int)

숫자 전용 상자입니다.

단, 소수점 없는 **정수**만 담을 수 있죠.

```
int year = 2025;
int count = 100;
```

**주의사항**: "소수점 있는 거 넣고 싶다?" &rarr; 안 됩니다. 그냥 숫자 똑 떨어지는 거만 가능해요.

## 실수형 변수 (double)

이번엔 소수점 있는 숫자도 담을 수 있는 상자예요.

예를 들어 "라면 물 550.5ml" 같은 거 저장하고 싶을 때 좋습니다.

```
double pi = 3.141592;
double weight = 72.5;
```

"아니 왜 이름이 double이에요?" 라고 묻는 분들 있는데…

옛날에 float라는 애가 있었거든요. 근데 얘보다 두 배(double) 더 정밀하게 저장할 수 있어서 그런 이름이 붙었어요. 그냥 역사적인 이유라 생각하시면 됩니다.

## 문자형 변수 (char)

특이한 상자입니다. 글자 딱 **한 글자**만 담을 수 있어요.

"이름의 첫 글자" 같은 거 보관할 때 쓰는 거죠.

- **글자는 꼭 작은따옴표 ' '로 감싸줘야 합니다.**

```
char first = 'A';
char gender = 'M';
```

참고로 여러 글자를 담고 싶으면 나중에 **문자열(string)** 배울 때 다룹니다.

char 하나로는 글자 하나만!

## printf()로 꺼내 쓰기

이제 상자에 데이터 넣었으니까 꺼내봐야죠.

그럴 때 쓰는 게 바로 `printf()` 함수예요.

근데 그냥 꺼내면 안 되고, "이 값은 정수입니다~" 하고 알려줘야 합니다.

그걸 해주는 게 **서식 지정자**예요.

### 필수 서식 지정자 3개

| 타입 | 서식 지정자 |
| --- | --- |
| int | %d |
| double | %lf |
| char | %c |

딱 이 세 가지만 알아도 90%는 커버됩니다.

## 예시 코드

```
#include 

int main() {
    int age = 25;
    double height = 175.5;
    char initial = 'H';

    printf("나이는 %d살이고, 키는 %lfcm입니다.\n", age, height);
    printf("이름의 첫 글자는 %c입니다.\n", initial);

    return 0;
}
```

실행하면 이렇게 나옵니다

```
나이는 25살이고, 키는 175.500000cm입니다.
이름의 첫 글자는 H입니다.
```

## 서식 지정자, 사실은 더 많다

여기까지가 기초인데… 사실 printf 서식 지정자는 종류가 엄청 많습니다.

- %u &rarr; 부호 없는 정수

- %f &rarr; 실수 출력 (double도 가능)

- %s &rarr; 문자열 출력

- %p &rarr; 메모리 주소 출력

- %x, %X &rarr; 16진수 출력

등등 줄줄이 있어요.

근데 지금 다 외우실 필요는 없습니다.

**일단 %d, %lf(또는 %f), %c만 잡고 나머지는 "필요할 때 찾는다"**

이게 현실적인 공부법이죠. 그게 더 기억에도 잘 남습니다.

## 오늘 정리

- 변수 = 데이터를 담는 상자

- 타입 = 상자의 종류

- printf() = 상자에서 꺼내서 보여주는 도구

- 처음엔 %d, %lf(%f), %c 세 가지만 알아도 충분
