## 1. 데이터 통신의 정의

**데이터(자료) + 통신(매개체) = 데이터 통신**

구 시대의 데이터 통신 : 유리병, 봉화, 비둘기

현 시대의 데이터 통신 : SNS, 메일, 메신저 등

### 01      데이터와 정보?

**데이터란?**

**의사,명령 등을 적절하게 전달하기위해 가공되지 않은 사실 및 지식(체온, 강우량, 기압)**

**정보란?**

**여러  데이터를 가공, 처리하여 만든 것(운동하기 좋은 날씨, 정상 체온, 폭우)**

### 02      통신이란?

**통신의 정의**

**거리가 떨어진 상태에서 수단이나 매체를 통해 정보를 교환하는 것**

### 03      데이터 통신이란?

**데이터 통신의 목적**

**컴퓨터의 발달(또는 네트워크 발달)과 보급으로 컴퓨터를 중심으로 데이터를 전송하기 위함**

- 아날로그 또는 디지털을 이용하여 데이터 교환 및 처리

- 컴퓨터와 단말기 또는 컴퓨터와 컴퓨터 간의 디지털 데이터 교환

- 데이터 처리 장치에 의해 처리된 또는 처리되는 정보의 전송

#### 데이터 통신의 이해

| 분류관점 | 통신의 종류 |
| --- | --- |
| 전송 채널 | 유선 통신,무선통신 |
| 송수신자의 이동여부 | 고정 통신,이동통신 |
| 신호 형태 | 아날로그통신(곡선형태), 디지털통신(직선블록형태) |
| 신호의 종류 | 전기통신,광통신(빛) |
| 이용대상 | 공중통신,전용통신(1:1) |
| 정보의 표현 형태 | 음성,데이터,화성,영상,멀티미디어 통신 |

## 2. 미래 지식 정보사회

### 01      미래 지식정보 사회

**미래 지식 정보 사회 (4차 산업혁명(정보화 사회))**

- **사용자 간의 끊김없는 상호작용으로 상호연결 : 참여,공유,개방**

- 사람과 사람, 사물과 사물, 기계와 기계, 사람과 기계 등 **모두가 생산자이면서 소비가 되는 새로운 디지털 생태계 조성**

- 네트워크, 서비스, 장치 등이 융합되는 **5G 시대로 빠르게 진화 중**

### 02      초연결 사회

**스마트폰 초소형 컴퓨터 같은 모바일 기기를 통해사람과 사물(IoT),동물(칩or센서),데이터,프로레스 등 모든것이 인터넷으로 연결된 사회(센서기반 네트워크 발달, 초고속 무선 인터넷 서비스)**

### 03      사물지능통신(M2M : 기계와 기계)

**사람 개입없이 센서 등을 통해 수집한 정보를 서로 주고받아 스스로 일을 처리**

- 모든 사물에 센서,통신 기능을 부과하여 지능적으로 **정보를 수집하여 상호 전달하는 네트워크**

- 모든 단편 정보를 다른 기기와 통신하며 우리 **주변 환경을 조절해주는 지능형 기술**

- M2M기술은 10년간 차곡차곡 정립이 되어왔음

- 자동차,의료,스마트 그린 기술(환경)

## 3. 데이터통신의 목표

**빠른 전달성, 정확성, 신속성**

### 01      데이터 전송의 정확성

> 데이터 전송 중 왜곡,잡음 등으로 인해 형태가 변형되어 정보가 잘못 전송되어서는 안됨

**정확성을 위해서는**

- 채널 코딩(에러를 줄여라)

- 에러 제어코딩

- 동기기술

- 스위칭 기술

- 흐름제어기술

### 02      데이터 전송의 효율성

> 획득 정보의 가치가 데이터 전송 장비의 가치보다 작으면 비효율

**효율성을 위한 기술로는**

- **소스코딩**(불필요한 정보를 줄여서 데이터의 효율성을 높임)

- 다중화 기술(멀티플래싱)

### 03      데이터 전송의 안정성(보안)

> 데이터의 내용이 제3자에게 누출되거나 변형되어 전달되어서는 안됨

안정성을 위한 기술로는

- secrecty coding
